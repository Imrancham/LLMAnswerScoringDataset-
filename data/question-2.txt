# Problem 2: Was ist eine Funktion?

# Rubriken

Rubrik Bereich: 0-3

Bewertungsskala:
0 Punkte: Keine relevante Antwort oder eine falsche Antwort (z. B. „Eine Funktion ist eine Aufgabe, die etwas erfüllt“).
1 Punkt: Gibt eine sehr einfache oder vage Definition (z. B. „Eine Funktion verbindet Dinge miteinander“).
2 Punkte: Nennt das Schlüsselkonzept einer Funktion richtig, aber nicht detailliert genug (z. B. „Eine Funktion ist eine Beziehung zwischen Eingaben und Ausgaben“).
3 Punkte: Liefert eine klare, detaillierte Definition (z. B. „Eine Funktion ist eine mathematische Beziehung, bei der jede Eingabe eine eindeutige Ausgabe hat, die oft als f(x) beschrieben wird“).
